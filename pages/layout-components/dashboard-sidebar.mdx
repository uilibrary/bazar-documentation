# Dashboard Sidebar

It's only used for vendor/admin dashboard.

import { Callout } from 'nextra-theme-docs'

<Callout>
  Folder location: `src/components/layouts/vendor-dashboard/DashboardSidebar`
</Callout>

## Examples

```jsx
import DashboardSidebar from './DashboardSidebar'

const Example = () => {
  return (
    <DashboardSidebar
      sidebarCompact={sidebarCompact}
      showMobileSideBar={showMobileSideBar}
      setSidebarCompact={handleCompactToggle}
      setShowMobileSideBar={handleMobileDrawerToggle}
    />
  )
}
```

## Used

- [VendorDashboardLayout](https://bazaar-doc.netlify.app/layouts/vendor-dashboard)

## Props

| Name                 |   Type   | Required | Default |
| :------------------- | :------: | -------: | ------- |
| sidebarCompact       | boolean  |      Yes | -       |
| showMobileSideBar    | boolean  |      Yes | -       |
| setSidebarCompact    | function |      Yes | -       |
| setShowMobileSideBar | function |      Yes | -       |
