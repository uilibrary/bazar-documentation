# Header

import { Callout } from 'nextra-theme-docs'

## Examples

```jsx
import Header from 'components/header/Header'
import SearchInput from 'components/search-box/SearchInput'
import SearchInputWithCategory from 'components/search-box/SearchInputWithCategory'

// example 1 - SaleLayout
const Example1 = () => {
  return <Header searchInput={<SearchInputWithCategory />} />
}

// example 2 - ShopLayout1
const Example2 = () => {
  return <Header isFixed={true} searchInput={<SearchInputWithCategory />} />
}

// example 3 - ShopLayout2
const Example3 = () => {
  return <Header isFixed={isFixed} searchInput={<SearchInput />} />
}
```

## Used

- [SaleLayout](https://bazaar-doc.netlify.app/layouts/sale-layout)
- [ShopLayout1](https://bazaar-doc.netlify.app/layouts/shop-layout-1)
- [ShopLayout2](https://bazaar-doc.netlify.app/layouts/shop-layout-2)

## Props

| Name        |     Type     | Required | Default |
| :---------- | :----------: | -------: | ------- |
| isFixed     |   boolean    | Optional | -       |
| className   |    string    | Optional | -       |
| searchInput | ReactElement |      Yes | -       |
