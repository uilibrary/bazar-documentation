# Translation

For Translation in this template use package

[Next i18next](https://github.com/i18next/next-i18next)

## Add New Laguage

- Add new locale **next-i18next.config.js** inside this file

```ts
module.exports = {
  i18n: {
    defaultLocale: 'en',
    locales: ['en', 'de'],
  },
}
```

- Create a folder inside public folder
- Follow their documentation - [Next i18next](https://github.com/i18next/next-i18next)
