# Navigations

There are 2 different navigations.

1. Navbar Navigation
2. Admin/Vendor Dashboard Navigation

## Navbar Navigation

- Edit **Navbar Navigation** style or design from `src/components/navbar/Navbar`
- Edit Navbar list data from `src/data/navbarNavigations`
- Inside `src/components/navbar` there are also 3 different **Menu**s
  - Mega Menu 1 - `src/components/navbar/MegaMenu`
  - Mega Menu 2 - `src/components/navbar/MegaMenu2`
  - Mobile Menu - `src/components/navbar/MobileMenu` for small device
